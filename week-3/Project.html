Project 1: Build a simple Node.js server with a basic API

In this project, you will create a simple Node.js server with a basic API that responds with a JSON object. The server will listen on a specific port and handle incoming requests to a specific route.

Instructions

    Step 1: Install Node.js and Create Project Directory

    First, ensure that Node.js is installed on your system. Node.js is a JavaScript runtime that allows you to run JavaScript code on the server side. You can download it from the official Node.js website.

    After installing Node.js, create a new directory for your project and navigate into it. This directory will hold all your project files.

    Initialize a new Node.js project using npm (Node Package Manager). This will create a package.json file in your project directory, which keeps track of your project dependencies and scripts.

    The -y flag automatically answers "yes" to all prompts, generating a default package.json file.

                
            node -v

            mkdir simple-web-server
            cd simple-web-server

            npm init -y

    Step 2: Install Express

                
            npm install express

    Step 3: Create Server File

    Create a file named server.js in your project directory and add the following code to set up the server and a simple API endpoint:

                
            const express = require('express');
            const path = require('path');
            const app = express();
            const PORT = 3000;

            // Serve static files from the "public" directory
            app.use(express.static(path.join(__dirname, 'public')));

            // Simple API endpoint
            app.get('/api/message', (req, res) => {
            res.json({ message: 'Hello from the API!' });
            });

            // Start the server
            app.listen(PORT, () => {
            console.log(`Server is running on http://localhost:${PORT}`);
            });

            
        
    Step 4: Create Public Directory and HTML File
    
        
    Create a directory named public in your project directory. Inside the public directory, create an index.html file with the following content:
    
        
                
                
                
                    
                        
                        
                        Simple Web Server with API
                        
                    
                    
                        
    Hello, World!
    
                        
    This is a simple web server using Node.js and Express.
    
                        
    API Message: Hello May!
    
                    
                
                
        
    
        
    Step 5: Run the Server
    
        
    Start the server by running the following command in your terminal:
    
        
                
            node server.js
            
        
    
        
    You should see a message indicating that the server is running:
    
        
                
            Server is running on http://localhost:3000
            
        
    
        
    Step 6: Open in Browser
    
        
    Open your web browser and navigate to http://localhost:3000. You should see the HTML content along with the message fetched from the API endpoint.
    
        
    Now you have a simple web server using Node.js and HTML with a basic API endpoint!

Project 2: Creating and Interacting with a Simple Database in JavaScript

Creating and interacting with a simple database in JavaScript can be done using various libraries and frameworks. One of the most popular and easy-to-use libraries for this purpose is SQLite with the sqlite3 package for Node.js. Let's walk through the steps to set up and interact with a SQLite database in a Node.js environment.

1. Setting Up the Environment

First, ensure you have Node.js installed on your system. You can download it from nodejs.org.

Next, create a new project directory and initialize it with npm:


mkdir mydatabase
cd mydatabase
npm init -y

Add .gitignore file:


touch .gitignore

Add code to .gitignore


/node_modules

Install the sqlite3 package:


npm install sqlite3

2. Creating and Interacting with the Database

Create a file named database.js and add the following code to it:


const sqlite3 = require('sqlite3').verbose();

// Connect to the database (or create it if it doesn't exist)
const db = new sqlite3.Database('mydatabase.db');

// Create tables
db.serialize(() => {
    db.run(`CREATE TABLE IF NOT EXISTS users (
        user_id INTEGER PRIMARY KEY AUTOINCREMENT,
        username TEXT NOT NULL,
        email TEXT NOT NULL,
        age INTEGER
    )`);

    db.run(`CREATE TABLE IF NOT EXISTS orders (
        order_id INTEGER PRIMARY KEY AUTOINCREMENT,
        user_id INTEGER,
        order_date TEXT,
        amount REAL,
        FOREIGN KEY (user_id) REFERENCES users(user_id)
    )`);
});

// Insert data
const insertData = () => {
    db.run(`INSERT INTO users (username, email, age) VALUES ('john_doe', 'john@example.com', 25)`);
    db.run(`INSERT INTO users (username, email, age) VALUES ('jane_smith', 'jane@example.com', 30)`);
    db.run(`INSERT INTO orders (user_id, order_date, amount) VALUES (1, '2023-07-01', 99.99)`);
    db.run(`INSERT INTO orders (user_id, order_date, amount) VALUES (2, '2023-07-02', 149.99)`);
};

// Read data
const readData = () => {
    db.all('SELECT * FROM users', (err, rows) => {
        if (err) {
            throw err;
        }
        console.log('Users:', rows);
    });

    db.all('SELECT * FROM orders WHERE user_id = 1', (err, rows) => {
        if (err) {
            throw err;
        }
        console.log('Orders for user_id = 1:', rows);
    });
};

// Update data
const updateData = () => {
    db.run(`UPDATE users SET email = 'john_new@example.com' WHERE username = 'john_doe'`);
    db.run(`UPDATE orders SET amount = 129.99 WHERE order_id = 2`);
};

// Delete data
const deleteData = () => {
    db.run(`DELETE FROM users WHERE username = 'john_doe'`);
    db.run(`DELETE FROM orders WHERE order_id = 1`);
};

// Function to close the database
const closeDatabase = () => {
    db.close((err) => {
        if (err) {
            return console.error(err.message);
        }
        console.log('Closed the database connection.');
    });
};

// Interact with the database
db.serialize(() => {
    insertData();
    readData();
    updateData();
    deleteData();
    readData();
    closeDatabase();
});

3. Running the Script

To run the script, execute the following command in your terminal:


node database.js

Explanation of the Code

Connecting to the Database: const db = new sqlite3.Database('mydatabase.db'); connects to a SQLite database file. If the file doesn't exist, it will be created.

Creating Tables: db.run is used to execute SQL commands to create the users and orders tables if they don't already exist.

Inserting Data: db.run inserts sample data into the users and orders tables.

Reading Data: db.all retrieves data from the users table and orders for a specific user from the orders table.

Updating Data: db.run updates specific records in the users and orders tables.

Deleting Data: db.run deletes specific records from the users and orders tables.

Closing the Database: db.close closes the database connection after all operations are completed.

By running this script, you can create a simple SQLite database, perform basic CRUD operations, and interact with the data using JavaScript.

Project 3: Connecting a Simple SQLite Database to an HTML Application

To connect a simple SQLite database to an HTML application and read/display data on a webpage, you'll need a server-side component to interact with the database and serve data to the client-side HTML/JavaScript. We'll use Node.js and Express for the server-side logic.

1. Setting Up the Environment

First, ensure you have Node.js and npm installed on your system.

Create a new project directory and initialize it with npm:


mkdir simple-database
cd simple-database
npm init -y

Add .gitignore file:


touch .gitignore

Add code to .gitignore


/node_modules

Install the necessary packages:


npm install express sqlite3

2. Setting Up the Server

Create a file named server.js and add the following code to set up the Express server and interact with the SQLite database:


const express = require('express');
const sqlite3 = require('sqlite3').verbose();
const path = require('path');

const app = express();
const db = new sqlite3.Database('mydatabase.db');

// Serve static files from the "public" directory
app.use(express.static('public'));

// API endpoint to get users data
app.get('/api/users', (req, res) => {
    db.all('SELECT * FROM users', (err, rows) => {
        if (err) {
            res.status(500).json({ error: err.message });
            return;
        }
        res.json({
            message: 'success',
            data: rows
        });
    });
});

// Start the server
const PORT = 3000;
app.listen(PORT, () => {
    console.log(`Server is running on http://localhost:${PORT}`);
});

Run this script:


node server.js

3. Setting Up the Database and Initial Data

In the same directory, create a script named setupDatabase.js to create the database tables and insert initial data:


const sqlite3 = require('sqlite3').verbose();

const db = new sqlite3.Database('mydatabase.db');

db.serialize(() => {
    db.run(`CREATE TABLE IF NOT EXISTS users (
        user_id INTEGER PRIMARY KEY AUTOINCREMENT,
        username TEXT NOT NULL,
        email TEXT NOT NULL,
        age INTEGER
    )`);

    db.run(`DELETE FROM users`); // Clear any existing data

    db.run(`INSERT INTO users (username, email, age) VALUES ('john_doe', 'john@example.com', 25)`);
    db.run(`INSERT INTO users (username, email, age) VALUES ('jane_smith', 'jane@example.com', 30)`);
});

db.close();

Run this script once to set up your database:


node setupDatabase.js
