Week 2 - Project

Build a simple HTML/CSS/Javascript page

In this project, you will create a simple webpage using HTML, CSS, and JavaScript. The webpage will have a header, main content area, and a button that changes the text content when clicked.

Instructions

  1. Create a new HTML file with the following structure:
    2.             
                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8" />
                    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
                    <title>My Simple Webpage</title>
                    <style>
                        /* Add your CSS styles here */
                        body {
                            font-family: Arial, sans-serif;
                            margin: 0;
                            padding: 15px;
                            background-color: #f0f0f0;
                        }
                        header {
                            background-color: #4CAF50;
                            color: white;
                            padding: 1em;
                            text-align: center;
                        }
                        main {
                            padding: 1em;
                            margin: 1em;
                            background-color: white;
                            border-radius: 8px;
                        }
                        button {
                            background-color: #4CAF50;
                            color: white;
                            border: none;
                            padding: 0.5em 1em;
                            text-align: center;
                            text-decoration: none;
                            display: inline-block;
                            font-size: 16px;
                            margin: 1em 0;
                            cursor: pointer;
                            border-radius: 4px;
                        }
                    </style>
                </head>
                <body>
                    <header>
                        <h1>Welcome to My Simple Webpage</h1>
                    </header>
                    <main>
                        <p id="greeting">Hello, world!</p>
                        <button onclick="changeGreeting()">Click me!</button>
                    </main>
                    <footer> 
                      <p> © 2024 My Simple Webpage. All rights reserved.</p> 
                  </footer> 
                    <script>
                        function changeGreeting() {
                            document.getElementById('greeting').innerText = "You clicked the button!";
                        }
                    </script>
                </body>
                </html>
  2. Save the file with a .html extension, for example, index.html.
  3. Open the file in a web browser to see your simple webpage in action.

Explanation

This simple webpage consists of a header with a welcome message, a main content area with a paragraph and a button. When the button is clicked, the text content of the paragraph changes to "You clicked the button!".

Feel free to expand on this project by adding more styles, HTML elements, or JavaScript functions to enhance the functionality and appearance of the webpage.
